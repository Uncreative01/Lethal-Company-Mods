Installing Lethal Company Mods in 10 steps.

## Step: 1 

Download the Repo ( Greeen Code Button > Download Zip )

## Step: 2

Run Lethal Company Once.

## Step: 3

In Steam:

Lethal Company > Right-Click > Manage > Browse Local Files.

## Step: 4

Add everything FROM the NOT THIS FOLDER inside the root of the Lethal Company Files ( Not the folder just everything inside )

## Step: 5

Run Lethal Company again ( There should command terminal on start-up )

## Step: 6 

Close Lethal Company.

## Step: 7

Add Mods:

Lethal Company > Right-Click > Manage > Browse Local Files > BepInEx > Plugins

## Step: 8

Add Mod files (They should mostly be .DLL files Except the 2 "Special ones")

## Step: 9

Start Lethal Company one more time, all mods should be on and ready to play.

## Step: 10

Join whoever is hosting. Fin
